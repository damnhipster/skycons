[Toolkit Skycons](http://skyglobal.github.io/toolkit-skycons/) 
========================

> A collection of brand approved icons for use across Sky.com

## Usage

### Include CSS in your own project

`bower install --save-dev pm-toolkit-skycons`

## Building Locally
